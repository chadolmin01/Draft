# 🎨 Gemini 디자인 개선 가이드

AI 스타트업 플랫폼 프론트엔드 디자인을 개선하는 방법

---

## 📋 서비스 컨셉 요약

**타겟**: 초보 창업자
**핵심 가치**: 원클릭으로 귀찮은 전체 업무를 해결
**차별점**: ChatGPT/Claude처럼 매번 프롬프트 짜야 하는 번거로움 없음

**디자인 방향성**:
- 구글 검색창처럼 심플하고 직관적
- 신뢰감을 주는 전문적인 느낌
- 혁신적이면서도 친근한 분위기
- 모던하고 깔끔한 UI

## 🎯 1단계: 디자인 시스템 생성

### Gemini에게 보여줄 파일

**파일**: `frontend/app/globals.css`

```css
[현재 globals.css 내용]
```

### Gemini 프롬프트 (복사해서 사용)

```
# 역할
당신은 최신 트렌드를 반영한 전문 UI/UX 디자이너입니다.

# 작업
이 CSS 파일의 디자인 시스템(색상 팔레트)을 개선해주세요.

# 서비스 특징
- AI 스타트업 아이디어 → 사업계획 자동 생성 플랫폼
- 타겟: 초보 창업자
- 컨셉: 심플하고 직관적, 전문적이면서 친근함
- 참고: 구글 검색창 스타일의 미니멀리즘

# 제약사항 (매우 중요!)
1. oklch 색상 포맷 유지 (현재 형식 그대로)
2. CSS 변수 이름 절대 변경 금지
3. 구조 변경 금지, 색상 값만 수정
4. :root와 .dark 모두 개선
5. 현대적이고 트렌디한 느낌
6. 접근성 고려 (충분한 대비)

# 출력 형식
- 수정된 globals.css 전체 코드
- 변경 사항 요약 (어떤 색상을 왜 선택했는지)

# 추가 요청
- Primary: 혁신적이고 신뢰감 있는 색상 (Indigo/Blue/Purple 계열)
- Accent: 활력 있는 포인트 컬러
- Background: 부드럽고 편안한 배경
```

## 🎨 2단계: 페이지별 디자인 개선

디자인 시스템이 완성되면 각 페이지를 하나씩 개선합니다.

### 2-1. 메인 입력 페이지

**파일**: `frontend/components/idea-input-page.tsx`

**Gemini 프롬프트**:

```
# 역할
당신은 최신 트렌드를 반영한 전문 UI/UX 디자이너입니다.

# 작업
이 React 컴포넌트의 디자인(UI/레이아웃)만 개선해주세요.

# 서비스 특징
- 구글 검색창처럼 심플하게 아이디어만 입력
- 티어 선택 (LIGHT/PRO/HEAVY)
- 첫인상이 중요한 메인 페이지

# 제약사항 (절대 지켜주세요!)
1. ❌ 기능 로직 절대 수정 금지
   - useState, useEffect, 이벤트 핸들러 내부 로직 변경 불가
   - 변수명, 함수명 변경 불가
2. ✅ 수정 가능한 것만
   - className (Tailwind CSS 클래스)
   - HTML 구조 (div, section 등의 레이아웃)
   - 텍스트 위치, 크기, 색상
   - 간격(padding, margin, gap)
   - 애니메이션 효과

# 디자인 방향
- 더 모던하고 트렌디하게
- 공간 활용 개선
- 시각적 계층 명확히
- 미묘한 애니메이션 추가 (hover, focus 등)
- 반응형 개선

# 출력 형식
- 수정된 전체 컴포넌트 코드
- 주요 변경 사항 요약
```

## ⚠️ 주의사항

### Gemini가 지켜야 할 것
- ✅ Tailwind CSS 클래스만 사용
- ✅ 기존 HTML 구조는 유지하되 레이아웃 개선 가능
- ✅ 애니메이션, 호버 효과 추가 가능
- ✅ 반응형 클래스 개선 가능

### Gemini가 절대 하면 안 되는 것
- ❌ useState, useEffect 수정
- ❌ 함수 로직 변경
- ❌ 변수명, 함수명 변경
- ❌ Props 타입 변경
- ❌ 새로운 라이브러리 추가
- ❌ inline style 사용 (Tailwind만 사용)

## 🎨 참고 디자인 시스템

현재 사용 중인 색상 (개선 전):
- Primary: 검은색 계열 (oklch(0.205 0 0))
- Background: 순백색 (oklch(1 0 0))
- Border: 밝은 회색 (oklch(0.922 0 0))

개선 방향:
- Primary: Indigo/Blue/Purple 계열로 변경
- Background: 약간 따뜻한 회색조
- Accent: 활력 있는 보조 색상 추가
